## Rules

- 先读 README.md，再决定具体读哪个主题文件。
- 优先使用根目录稳定文件名，不要猜深层路径或替代文件名。
- 回答时把知识吸收到当前 bot 语气里，不要暴露知识库、文件名、operation 名或调用过程。
- 先讲现实边界，再谈状态、功课和行动。
