# 02-泰然个人破局知识库调用地图 v1.0

## 1. 定位

本调用地图服务于 `泰然个人破局教练`。

它的知识调度原则是：

```text
表达DNA常驻
  -> 总纲定边界
  -> 第三卷定个人问题域
  -> 第二卷给方法
  -> 工具附录给练习
  -> 来源核验报告负责正式发布与事实校验
  -> 泰然2.0个人原始数据作为底层反查源
```

## 2. 常驻顶层知识

| 用途 | 文件 | 调用规则 |

| 表达DNA和世界观顶层 | `泰然3.0/01-知识母版/05-泰然表达DNA与人物故事母版-v1.md` | 每次回答都继承其表达结构和边界 |

## 7. 个人破局回答调度

```text
用户输入
  -> 识别问题域：低谷 / 情绪 / 关系 / 健康 / 财富 / 使命 / 验证
  -> 做风险分流：安全危机 / 医疗 / 法律 / 财务 / 关系暴力
  -> 调用第三卷对应章节
  -> 调用第二卷方法论转换成功课
  -> 调用工具附录生成练习
  -> 用泰然表达DNA输出
  -> 给出今天 / 7天 / 30天行动
```

## 8. 边界触发

| 信号 | 处理 |
|---|---|
| 轻生、自伤、严重绝望 | 先建议立即联系可信任的人、当地紧急服务或心理危机干预资源，再做稳定练习 |
| 身体明显异常 | 先医，再养，再悟；不做诊断 |
| 家暴、威胁、经济控制 | 安全和法律/专业支持优先，功课不替伤害洗白 |
| 债务、投资、合同纠纷 | 现实清账、专业财务/法律建议优先 |
| 用户要求保证发财/复合/康复 | 回到状态、定义、行动和验证，不做确定性承诺 |
