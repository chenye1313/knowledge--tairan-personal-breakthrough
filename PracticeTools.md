## 附录A 三大功课一页练习表

### 使用说明

这份表不是为了把生命变成表格，而是在情绪最乱、事情最急、头脑最容易掉回旧剧情时，给自己一个可落脚的顺序。

```text
先明心见性，看见卡点。
再神游八荒，打开视角。
最后逍遥有为，落到行动。
```

不要一开始就问“我该怎么办”。很多时候，人之所以不知道怎么办，是因为还没看清自己卡在哪里。也不要只停在“我看见了”。看见之后没有新的定义、新的选择、新的行动，功课还没有落地。

### A1. 明心见性

```text
事件：
涉及的人：
发生的时间：
我当时最强的情绪：
我脑子里自动编出的故事：
我把自己放在什么角色：
  受害者 / 拯救者 / 控制者 / 讨好者 / 证明者 / 被抛弃的人 / 必须负责所有人的人
如果只看事实，不加解释，这件事可以怎么描述：
这件事照见我对____的依赖：
这件事照见我对____的控制：
这件事照见我在____上的匮乏：
这件事照见我一直想证明____：
这件事照见我害怕____：
这件事照见我不愿面对____：
```

边界提醒：填空不是自责。如果一段关系里有暴力、欺骗、严重伤害，填空不是说“都是我吸引来的，所以我活该”。更准确的口径是：外在责任归外在责任，我的功课归我的功课，边界归边界，行动归行动。

### A2. 抽离

```text
如果我是旁观者，我会看到什么：
如果这件事发生在朋友身上，我会怎么提醒他：
如果十年后的我看今天，他会怎么理解：
如果这是人生游戏里的一个关卡，这一关叫什么：
我现在有情绪，但我不等于这个情绪。我可以站到的新位置是：
```

抽离不是冷漠，也不是压抑情绪。抽离只是从“我就是这个情绪”里退一步。有空间，才有选择。

### A3. 感恩

感恩不是替坏事洗白。感恩的意思是：我不再让这件事只负责伤害我，我要从里面拿回成长材料。

```text
这件事提醒我：
它让我看见的旧模式：
它逼我面对的事实：
它提醒我需要建立的边界：
它让我长出的能力：
它让我重新理解的关系：
我感谢自己没有继续沉睡在：
```

有些人暂时做不到感谢对方，这是正常的。可以先从感谢自己愿意面对开始。真实比漂亮更重要。

### A4. 重塑

```text
过去我把这件事定义为：
现在我愿意把它重新定义为：

过去我把自己定义为：
现在我愿意把自己重新定义为：

过去我期待对方：
现在我把注意力收回到我可以：

过去我用____反应。
现在我选择用____行动。
```

行动落地：

```text
接下来24小时内，我要做的一件小事：
接下来7天内，我要停止的一个旧反应：
接下来30天内，我要建立的一个新习惯：
我需要向谁表达：
我需要向谁道歉：
我需要向谁感谢：
我需要设置什么边界：
我需要承担什么责任：
我需要寻求什么专业支持：
```

### A5. 神游八荒

时间神游：

```text
如果十年后的我回看今天，他会怎么理解这件事：
如果临终前的我回看今天，他会觉得什么重要：
如果这是我人生某一章的开头，这章标题是什么：
如果这件事不是结局，而是转折，它在把我推向哪里：
```

人物神游：

```text
我选择的人物/未来自己/长辈/老师：
如果他面对这件事，会先看什么：
他不会被什么困住：
他会提醒我承担什么：
他会让我放下什么：
他会让我立刻做什么：
```

家族神游：

```text
如果祖先看见今天的我，他会希望我怎么做：
如果孩子将来回看今天，他会从我身上学到什么：
如果我是家族时间链中的一环，我现在的选择会给后面的人留下什么：
我是在延续家族旧伤，还是开启新的程序：
```

自然神游：

```text
如果我是山，我会如何看这件事：
如果我是水，我会如何穿过这个阻碍：
如果我是树，我现在应该扎根，还是开枝：
如果我是风，我需要带走什么旧气：
如果我是土地，我可以承载什么：
```

### A6. 逍遥有为

```text
现在我的身体是紧的还是松的：
我的心是开着的还是关着的：
我是在害怕里行动，还是在清明里行动：
我是在证明自己，还是在创造价值：
此刻最小但正确的一步是什么：
我可以整理哪一个空间：
我可以完成哪一个拖了很久的小任务：
我可以给谁发一条真诚的信息：
我可以停止哪一种内耗：
如果短期没有结果，我还做不做：
如果没有人夸我，我还愿不愿意创造：
我不再等待什么：
```

### A8. 使用边界

功课不是万能解释，不是压抑情绪，不是替伤害洗白，不是表演给别人看。严重抑郁、创伤、暴力关系、法律纠纷、重大财务风险、医疗问题和人身安全问题，应优先寻求专业支持。向内求不是孤立无援，而是在拿回主导权之后，更清醒地选择需要的外部帮助。
