# 来源分层与可信度规则 v1

## 1. 为什么要分层

泰然材料包含原始录制、分析报告、课程分析、已有整理、AI克隆提示词和外部参考。不同材料不能混用同一权重，否则会把“泰然老师原话”“整理者推断”“AI封装话术”混成一团。

## 2. 来源等级

| 等级 | 类型 | 可信用途 | 使用限制 |
|---|---|---|---|
| A | 原始录制、原话、课程实录 | 定义核心思想、故事、案例、语言风格 | 需要去噪，区分闲聊、访谈、正式表达 |
| B | 泰然1.0分析报告、线下课分析 | 快速识别结构、概念和行为模式 | 需要回查原始材料，不可直接当最终事实 |
| C | 已有泰然整理 | 可继承目录、台账、概念表、蓝图 | 需要审计是否遗漏人物源起和边界声明 |
| D | GPT封装、AI行为法典 | 用于智能体人设、表达风格、对话规则 | 不可作为泰然老师思想源头 |
| E | 外部参考资料 | 解释传统文化、家族、风水、号码背景 | 只能辅助，不能替代泰然老师本体 |

## 3. 标注规则

后续每个知识单元建议标注：

```text
知识点：
来源等级：A/B/C/D/E
来源文件：
证据类型：原话/案例/分析/推断/外部参考
可靠度：高/中/低
使用场景：书籍/课程/智能体/内部索引
边界提醒：
```

## 4. 初始可信度原则

1. 凡是涉及泰然老师个人经历，优先使用原始录制。
2. 凡是涉及方法结构，可用分析报告先建框架，再回查原文。
3. 凡是涉及风水、号码、能量、频率，必须加“体系内口径”和“不作确定性承诺”的边界。
4. 凡是用于对外成书，不能只引用AI封装提示词。
5. 凡是用于智能体，可以引用行为法典，但要标注其为封装层。
